Get the last shared links for your Geekli.st account

== Description ==

In a Widget show the shared links in your Geekli.st account

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `wp-geeklist-links-reader` folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.
3. Enter your API credentials. (http://geekli.st/applications)

= Usage =

Drag and Drop the Widget into your selected area
